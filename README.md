# Paçocas do Gustavo

Site institucional desenvolvido para apresentar o trabalho de Gustavo Santos Martins, vendedor de paçocas na região de Santo Amaro, São Paulo.

O objetivo do projeto é permitir que clientes conheçam sua história, rotina, objetivos e formas de apoio através de um QR Code divulgado durante as vendas.

## Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Funcionalidades

* Navbar com navegação entre seções
* Hero Section com carousel de imagens
* Apresentação da história do vendedor
* Seção sobre a rotina de trabalho
* Objetivos e metas futuras
* Informações sobre como cada compra ajuda
* Estatísticas e curiosidades
* Área de contato e apoio
* QR Code para divulgação
* Contadores animados com JavaScript
* Botão de copiar chave PIX
* Botão de voltar ao topo
* Scroll suave entre seções

## Objetivo Social

O projeto busca aproximar clientes e apoiadores da realidade do trabalho realizado por Gustavo, mostrando como a venda de paçocas contribui para custear despesas do dia a dia e auxiliar na conquista de seus objetivos acadêmicos e profissionais.
